# 🚀 Krishal Poudel — Personal Portfolio

> A premium, futuristic personal portfolio website built with **pure HTML, CSS, and JavaScript** — no build step, no frameworks, no Node.js required.

🌐 **Live**: http://krishalpoudel.com.np/

---

## ✨ Features

- 🎬 **Animated loader** with hexagonal "KP" logo
- 🎯 **Custom cursor** (dot + ring) that adapts on hover (desktop)
- 🌌 **Particle network background** with mouse interaction
- 🚀 **Hero section** with hexagonal portrait, typing role animation, glow rings
- 👤 **About section** with floating cards and animated stat counters
- 🛠️ **Skills grid** — 8 themed cards with unique color glows + 3D tilt
- 🧠 **Core Learning Areas** — AI & ML, Financial Markets, Data Analysis
- 💼 **Projects** — 6 experiment cards with live status indicators
- 💭 **Philosophy** quote in elegant serif typography
- 📅 **Vertical Timeline** — Trading (2024), Python/Prompt/Algo (2025), Future
- 📬 **Contact form** with mailto integration
- 📱 **Fully responsive** with mobile menu
- 🎨 **Premium dark + cyan/purple gradient theme**

## 🚀 How to Run Locally

### Option 1 — Open Directly (Easiest)
Just **double-click `index.html`** — opens in your browser instantly! ✨

### Option 2 — Local Server (Recommended)

**Python:**
```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

**Node.js:**
```bash
npx serve .
```

**VS Code:** Install "Live Server" extension → right-click `index.html` → "Open with Live Server"

## 🌐 Deploy Anywhere

This is a **pure static site** — drop it on any hosting platform:

| Platform | How |
|----------|-----|
| **Cloudflare Pages** | Dashboard → Pages → Create with direct upload → drag the folder |
| **GitHub Pages** | Settings → Pages → Source: `main` branch → `/ (root)` |
| **Netlify** | Drag & drop the folder at [app.netlify.com/drop](https://app.netlify.com/drop) |
| **Vercel** | `vercel deploy` or drag in dashboard |
| **Any web host** | Upload all files via FTP/cPanel preserving folder structure |

### Quick Deploy to GitHub Pages
1. Go to repo **Settings** → **Pages**
2. **Source**: Deploy from a branch
3. **Branch**: `main` / `/ (root)`
4. **Save** — site live in ~1 minute at `https://krishal-tech.github.io/Personal-Protfolio/`

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom animations, glassmorphism, gradients
- **JavaScript** — Vanilla ES6, no frameworks
- **Tailwind CSS** (CDN) — Utility classes
- **Font Awesome 6** (CDN) — Icons
- **AOS** (CDN) — Scroll-triggered animations
- **VanillaTilt** (CDN) — 3D card hover effects
- **Google Fonts** — Space Grotesk, Cormorant Garamond, JetBrains Mono

> All libraries load via CDN — no installation needed. Just open and go!

## 🎨 Customization

| To Change | Edit File |
|-----------|-----------|
| Text content (name, descriptions) | `index.html` |
| Colors / fonts / spacing | `assets/css/style.css` (look for `:root` variables at the top) |
| Animations / interactions | `assets/js/app.js` |
| Profile photo | Replace `assets/images/krishal-profile.jpg` |

### Theme Colors (in `style.css`)
```css
--accent-cyan: #22d3ee;
--accent-purple: #a855f7;
--accent-pink: #ec4899;
--bg-primary: #07090f;
```
